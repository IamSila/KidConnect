About kidConnect

project kidConnect

A system designed for finding lost children. It focuses on creating a safe, approachable, and innovative environment, leveraging technology to reunite children with their families. The design emphasizes connection, trust, and child-friendly features with vibrant, warm, and engaging elements. Features inlcude:

  • Report a missing child.
  • Query from the database to see reported children and their found/pending status
  • Send messages to parent/guardians if you have relevan information.

We appreciate support to make this project better
